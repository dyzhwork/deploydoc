# Scss 函数-混入

场景 1,我们需要做一个居中效果,比如外层容器类名为 out_box1,内层容器类名为 inner_box1,那我们使用弹性盒实现,html 代码如下:

<div class="out_box1">
  <div class="inner_box1"></div>
</div>
1
2
3

CSS 代码如下:

.out_box1 {
  display: flex;
  align-items: center;
  justify-content: center;
}
1
2
3
4
5

但是如果有多个容器都会使用居中这个样式,比如下方代码,out_box2 也需要设置居中样式,那我们可以这样做,使用 Scss 的混合:

<div class="out_box1">
  <div class="inner_box1"></div>
</div>
<div class="out_box2">
  <div class="inner_box2"></div>
</div>
1
2
3
4
5
6
@mixin flexCenter {
  display: flex;
  align-items: center;
  justify-content: center;
}
.out_box1 {
  @include flexCenter;
}
.out_box2 {
  @include flexCenter;
}

  display: flex;
  align-items: $layout;
  justify-content: $layout;

.out_box1 {
  @include flex(center);
}
.out_box2 {
  @include flex(start);
